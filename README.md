PyPiMenu
========

PyPiMenu is an small application (written in python) for controll some stuff with the 16x2 Adafruit LCD-Module and its 5 buttons.
I just started the developement there is not realy much working yet.


Requirements:
- RaspberryPi with Rasbian (not tested for other OS)
- Python (V2.7 - default)
- MPDaemon/MPClient (MPD must be running yet)

Features (in Work) :
- controll mpd for local music and internet radio
- shutdown/ reboot the Pi
- standby (LCD turns off after time)
- show LAN and internet IP-adress
